Net Skip은 일상적인 웹 브라우징에 명료함과 빠름을 가져다주는, 사려 깊게 설계된 웹 브라우저입니다. 무료이고 오픈소스이며, 당신의 기기에서 빠르고 수월하게 느껴지도록 만들어졌습니다.

Android의 네이티브 웹 엔진을 기반으로 하여 Net Skip은 빠르게 실행되고 페이지를 부드럽게 렌더링합니다. 인터페이스는 읽는 동안 조용히 물러납니다. 탭은 한 번의 제스처로 서로 전환되고, 스크롤하면 도구 모음은 시야에서 사라지며, 초점은 마땅히 있어야 할 곳에 머무릅니다 — 바로 앞의 페이지에.

주요 기능

• 네이티브 웹 엔진이 구현하는 빠른 페이지 로딩과 부드러운 스크롤
• 읽는 동안 물러나는 깔끔하고 정돈된 인터페이스
• 한 번의 제스처로 탭 사이를 전환
• 열려 있는 모든 페이지의 실시간 미리보기를 보여주는 시각적 탭 개요
• 자동 세션 복원 — 탭은 그대로 그 자리에 있습니다
• 기본적으로 활성화된 광고 및 추적기 차단 기능 내장
• 입력 중 제안이 표시되는 검색 엔진 선택
• 페이지 내 찾기, 조절 가능한 텍스트 확대/축소, 데스크톱 사이트 요청 옵션
• 즐겨찾기와 전체 방문 기록, 원하는 항목을 간단하게 지울 수 있는 컨트롤
• 라이트, 다크, 시스템 설정에 따른 외관 모드
• 보다 촉각적인 경험을 위한 선택적 햅틱 피드백
• 고급 사용자 위한 JavaScript 및 사용자 지정 user-agent 설정

설계에서부터 보호되는 프라이버시

Net Skip은 단순한 생각을 중심으로 설계되었습니다. 당신의 브라우징은 당신의 것입니다.

• 어떠한 광고도 없음
• 추적, 분석, 텔레메트리 없음
• 인앱 결제 또는 구독 없음
• 계정 없음, 로그인 없음, 프로필 없음

Net Skip에서 일어나는 일은 당신의 기기에 머무릅니다.

무료이며 오픈소스

Net Skip은 다운로드도 사용도 무료이며, 프리미엄 등급이나 잠금 해제할 것은 전혀 없습니다. 전체 소스 코드는 누구나 읽고, 검토하고, 기여할 수 있도록 공개되어 있습니다. 프로젝트는 https://github.com/Net-Skip 에서 확인할 수 있습니다.
